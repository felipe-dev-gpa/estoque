COMO USAR

ACESSANDO A PAGINA INICIAL

Quando o usuário abre o aplicativo, ele será direcionado para a página inicial que contém:

LOGO DO ESTOQUEMAIS

Botões de Navegação: Há botões para navegar entre diferentes partes do aplicativo, como a tela inicial, formulário de cadastro de produtos e exportação para Excel.

BOTÃO DE VOLTAR

Este botão permite que o usuário retorne à tela anterior ou à tela inicial, dependendo de onde ele está dentro do aplicativo.

BOTÃO DE EXPORTAÇÃO EM PLANILHA

Se o usuário precisar exportar os dados do inventário para um arquivo Excel, ele pode clicar neste botão. Essa funcionalidade é útil para gerar relatórios ou armazenar as informações fora do aplicativo.

BOTÃO ADICIONAR PRODUTO

O usuário é direcionado para a página do formulário de cadastro de produto, onde ele pode preencher informações sobre um novo item de estoque.
